## Decisão de Design para Este Projeto

### Filosofia Escolhida: **Modern Minimalism com Glassmorphism 2.0**

Combinaremos:
- **Minimalismo moderno** como base (limpo, profissional, sem distrações)
- **Glassmorphism 2.0** para elementos de interface (painéis translúcidos)
- **Micro-delight interactions** para feedback suave
- **Organic layouts** assimétricos em certas seções
- **Bold typography** para storytelling
- **Dark mode** como padrão com toggle light/dark
- **Animações suaves** scroll-triggered

### Paleta de Cores
- **Primária**: Azul profundo (confiança, profissionalismo)
- **Secundária**: Branco/Cinza claro (minimalismo)
- **Acentos**: Verde menta ou roxo suave (modernidade)
- **Dark Mode**: Cinza escuro com acentos claros

### Tipografia
- **Display**: Fonte moderna e bold (ex: Poppins Bold)
- **Body**: Fonte legível e limpa (ex: Inter, Outfit)
- **Hierarquia**: Contraste forte entre títulos e corpo

### Componentes Principais
1. **Navigation** - Minimalista, sticky, com animação suave
2. **Hero Section** - Glassmorphism com imagem de fundo
3. **About Section** - Layout assimétrico com imagem e texto
4. **Projects Gallery** - Grid com hover effects e animações
5. **Skills Section** - Visualização moderna de habilidades
6. **Contact Section** - Formulário limpo com micro-interactions
7. **Footer** - Minimalista com links sociais

### Animações
- Fade-in on scroll
- Hover effects suaves em cards
- Micro-interactions em botões
- Parallax suave em hero
- Transições de página suaves
